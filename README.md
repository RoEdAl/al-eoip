# Kernel mode EoIP (Ethernet Over IP) tunnel compatible with MikroTik RouterOS [Arch Linux packages]

Two packages:
* [`eoip-dkms`](eoip-dkms) - kernel modules,
* [`eoip-utils`](eoip-utils) - management utilities.
